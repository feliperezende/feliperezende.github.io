Barrier is an open-source kvm software.

- Install on macOS
{% highlight shell %}
brew cask install barrier
{% endhighlight %}

---

- Install on Manjaro Linux
{% highlight shell %}
pacman -S barrier
{% endhighlight %}

Configure server and clients.
