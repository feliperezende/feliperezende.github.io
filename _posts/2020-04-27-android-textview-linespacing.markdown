You can use **lineSpacingExtra** or **lineSpacingMultiplier** in your XML file.


## Example:

{% highlight xml %}

<TextView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:lineSpacingMultiplier="0.7">

{% endhighlight %}
